# 高中知识点集合

高中语文·数学·英语·物理·化学·生物六科知识点查询网站，涵盖高考核心考点。

## 内容
- **化学** — 元素化合物、反应原理、化学实验、物质结构、有机化学
- **数学** — 导数、解析几何、三角函数、立体几何、统计概率
- **物理** — 运动学力学、电学、磁学、热学光学、原子物理
- **生物** — 遗传与进化、生物技术与工程、分子与细胞
- **英语** — 高频词汇、语法专题、应用文写作、读后续写
- **语文** — 文言文 诗词 成语 阅读技巧

## 技术
纯静态 HTML 单文件，无框架依赖，支持暗色模式、全文搜索、响应式布局。
