《劝学》（节选）-- 荀子

君子曰：学不可以已。

青，取之于蓝，而青于蓝；冰，水为之，而寒于水。木直中(zhòng)绳，輮(róu)以为轮，其曲中规。虽有槁暴(pù)，不复挺者，輮使之然也。故木受绳则直，金就砺(lì)则利，君子博学而日参(cān)省(xǐng)乎己，则知(zhì)明而行无过矣。

吾尝终日而思矣，不如须臾(yú)之所学也。吾尝跂(qì)而望矣，不如登高之博见也。登高而招，臂非加长也，而见者远；顺风而呼，声非加疾也，而闻者彰。假舆(yú)马者，非利足也，而致千里；假舟楫(jí)者，非能水也，而绝江河。君子生(xìng)非异也，善假于物也。

积土成山，风雨兴焉；积水成渊，蛟龙生焉；积善成德，而神明自得，圣心备焉。故不积跬(kuǐ)步，无以(yǐ)至千里；不积小流，无以成江海。骐骥(qí jì)一跃，不能十步；驽(nú)马十驾，功在不舍。锲(qiè)而舍之，朽木不折；锲而不舍，金石可镂(lòu)。蚓无爪牙之利，筋骨之强，上食埃土，下饮黄泉，用心一也。蟹六跪而二螯(áo)，非蛇鳝(shàn)之穴无可寄托者，用心躁也。

《师说》-- 韩愈

古之学者必有师。师者，所以传道受业解惑也。人非生而知之者，孰能无惑？惑而不从师，其为惑也，终不解矣。生乎吾前，其闻道也固先乎吾，吾从而师之；生乎吾后，其闻道也亦先乎吾，吾从而师之。吾师道也，夫庸知其年之先后生于吾乎？是故无贵无贱，无长无少，道之所存，师之所存也。

嗟乎！师道之不传也久矣！欲人之无惑也难矣！古之圣人，其出人也远矣，犹且从师而问焉；今之众人，其下圣人也亦远矣，而耻学于师。是故圣益圣，愚益愚。圣人之所以为圣，愚人之所以为愚，其皆出于此乎？爱其子，择师而教之；于其身也，则耻师焉，惑矣。彼童子之师，授之书而习其句读(dòu)者，非吾所谓传其道解其惑者也。句读之不知，惑之不解，或师焉，或不(fǒu)焉，小学而大遗，吾未见其明也。巫医乐师百工之人，不耻相师。士大夫之族，曰师曰弟子云者，则群聚而笑之。问之，则曰："彼与彼年相若也，道相似也，位卑则足羞，官盛则近谀(yú)。"呜呼！师道之不复，可知矣。巫医乐师百工之人，君子不齿，今其智乃反不能及，其可怪也欤(yú)！

圣人无常师。孔子师郯(tán)子、苌(cháng)弘、师襄(xiāng)、老聃(dān)。郯子之徒，其贤不及孔子。孔子曰：三人行，则必有我师。是故弟子不必不如师，师不必贤于弟子，闻道有先后，术业有专攻，如是而已。

李氏子蟠(pán)，年十七，好古文，六艺经传(zhuàn)皆通习之，不拘于时，学于余。余嘉其能行古道，作《师说》以贻(yí)之。

《赤壁赋》-- 苏轼

壬戌(rén xū)之秋，七月既望，苏子与客泛舟游于赤壁之下。清风徐来，水波不兴。举酒属(zhǔ)客，诵明月之诗，歌窈窕(yǎo tiǎo)之章。少焉，月出于东山之上，徘徊于斗(dǒu)牛之间。白露横江，水光接天。纵一苇之所如，凌万顷之茫然。浩浩乎如冯(píng)虚御风，而不知其所止；飘飘乎如遗世独立，羽化而登仙。

于是饮酒乐甚，扣舷(xián)而歌之。歌曰："桂棹(zhào)兮兰桨，击空明兮溯(sù)流光。渺渺兮予怀，望美人兮天一方。"客有吹洞箫者，倚歌而和(hè)之。其声呜呜然，如怨如慕，如泣如诉，余音袅袅(niǎo)，不绝如缕。舞幽壑之潜蛟，泣孤舟之嫠(lí)妇。

苏子愀(qiǎo)然，正襟危坐而问客曰："何为其然也？"客曰："'月明星稀，乌鹊南飞'，此非曹孟德之诗乎？西望夏口，东望武昌，山川相缪(liáo)，郁乎苍苍，此非孟德之困于周郎者乎？方其破荆州，下江陵，顺流而东也，舳舻(zhú lú)千里，旌旗蔽空，酾(shī)酒临江，横槊(shuò)赋诗，固一世之雄也，而今安在哉？况吾与子渔樵(qiáo)于江渚(zhǔ)之上，侣鱼虾而友麋(mí)鹿，驾一叶之扁(piān)舟，举匏樽(páo zūn)以相属。寄蜉蝣(fú yóu)于天地，渺沧海之一粟。哀吾生之须臾，羡长江之无穷。挟飞仙以遨游，抱明月而长终。知不可乎骤得，托遗响于悲风。"

苏子曰："客亦知夫(fú)水与月乎？逝者如斯，而未尝往也；盈虚者如彼，而卒莫消长也。盖将自其变者而观之，则天地曾(céng)不能以一瞬；自其不变者而观之，则物与我皆无尽也，而又何羡乎！且夫天地之间，物各有主，苟非吾之所有，虽一毫而莫取。惟江上之清风，与山间之明月，耳得之而为声，目遇之而成色，取之无禁，用之不竭，是造物者之无尽藏(zàng)也，而吾与子之所共适。"

客喜而笑，洗盏更酌。肴核既尽，杯盘狼籍。相与枕藉(jiè)乎舟中，不知东方之既白。

《短歌行》-- 曹操

对酒当歌，人生几何！譬如朝露，去日苦多。

慨当以慷，忧思难忘。何以解忧？唯有杜康。

青青子衿(jīn)，悠悠我心。但为君故，沉吟至今。

呦呦(yōu)鹿鸣，食野之苹。我有嘉宾，鼓瑟吹笙。

明明如月，何时可掇(duō)？忧从中来，不可断绝。

越陌度阡，枉用相存。契阔谈讌(yàn)，心念旧恩。

月明星稀，乌鹊南飞。绕树三匝(zā)，何枝可依？

山不厌高，海不厌深。周公吐哺(bǔ)，天下归心。

《归园田居·其一》-- 陶渊明

少无适俗韵，性本爱丘山。

误落尘网中，一去三十年。

羁鸟恋旧林，池鱼思故渊。

开荒南野际，守拙归园田。

方宅十余亩，草屋八九间。

榆柳荫后檐，桃李罗堂前。

暧暧(ài)远人村，依依墟里烟。

狗吠深巷中，鸡鸣桑树颠。

户庭无尘杂，虚室有余闲。

久在樊(fán)笼里，复得返自然。

《梦游天姥吟留别》-- 李白

海客谈瀛洲，烟涛微茫信难求；越人语天姥(mǔ)，云霞明灭或可睹。天姥连天向天横，势拔五岳掩赤城。天台四万八千丈，对此欲倒东南倾。

我欲因之梦吴越，一夜飞度镜湖月。湖月照我影，送我至剡(shàn)溪。谢公宿处今尚在，渌(lù)水荡漾清猿啼。脚著谢公屐(jī)，身登青云梯。半壁见海日，空中闻天鸡。千岩万转路不定，迷花倚石忽已暝。熊咆龙吟殷(yǐn)岩泉，栗深林兮惊层巅。云青青兮欲雨，水澹澹(dàn)兮生烟。列缺霹雳，丘峦崩摧。洞天石扉，訇(hōng)然中开。青冥浩荡不见底，日月照耀金银台。霓为衣兮风为马，云之君兮纷纷而来下。虎鼓瑟兮鸾回车，仙之人兮列如麻。忽魂悸以魄动，恍惊起而长嗟。惟觉(jiào)时之枕席，失向来之烟霞。

世间行乐亦如此，古来万事东流水。别君去兮何时还？且放白鹿青崖间，须行即骑访名山。安能摧眉折腰事权贵，使我不得开心颜？

《登高》-- 杜甫

风急天高猿啸哀，渚(zhǔ)清沙白鸟飞回。

无边落木萧萧下，不尽长江滚滚来。

万里悲秋常作客，百年多病独登台。

艰难苦恨繁霜鬓(bìn)，潦倒新停浊酒杯。

《念奴娇·赤壁怀古》-- 苏轼

大江东去，浪淘尽，千古风流人物。故垒西边，人道是：三国周郎赤壁。乱石穿空，惊涛拍岸，卷起千堆雪。江山如画，一时多少豪杰。

遥想公瑾当年，小乔初嫁了，雄姿英发。羽扇纶(guān)巾，谈笑间，樯橹(qiáng lǔ)灰飞烟灭。故国神游，多情应笑我，早生华发。人生如梦，一尊还酹(lèi)江月。

《永遇乐·京口北固亭怀古》-- 辛弃疾

千古江山，英雄无觅，孙仲谋处。舞榭(xiè)歌台，风流总被、雨打风吹去。斜阳草树，寻常巷陌，人道寄奴曾住。想当年，金戈铁马，气吞万里如虎。

元嘉草草，封狼居胥(xū)，赢得仓皇北顾。四十三年，望中犹记，烽火扬州路。可堪回首，佛(bì)狸祠下，一片神鸦社鼓。凭谁问，廉颇老矣，尚能饭否？

《声声慢》-- 李清照

寻寻觅觅，冷冷清清，凄凄惨惨戚戚。乍暖还寒时候，最难将息。三杯两盏淡酒，怎敌他、晚来风急！雁过也，正伤心，却是旧时相识。

满地黄花堆积，憔悴损，如今有谁堪摘？守着窗儿，独自怎生得黑！梧桐更兼细雨，到黄昏、点点滴滴。这次第，怎一个愁字了得！

《虞美人》-- 李煜

春花秋月何时了？往事知多少。小楼昨夜又东风，故国不堪回首月明中。

雕栏玉砌应犹在，只是朱颜改。问君能有几多愁？恰似一江春水向东流。

《琵琶行》（节选背诵段落）-- 白居易

千呼万唤始出来，犹抱琵琶半遮面。转轴拨弦三两声，未成曲调先有情。

大弦嘈嘈如急雨，小弦切切如私语。嘈嘈切切错杂弹，大珠小珠落玉盘。

别有幽愁暗恨生，此时无声胜有声。

同是天涯沦落人，相逢何必曾相识！

座中泣下谁最多？江州司马青衫湿。

（二）18个文言虚词详解

高考常考18个虚词：而、何、乎、乃、其、且、若、所、为、焉、也、以、因、于、与、则、者、之

1. 而
连词：并列又、和蟹六跪而二螯
连词：承接就、接着故木受绳则直，金就砺则利
连词：递进而且君子博学而日参省乎己
连词：转折但是、却青，取之于蓝，而青于蓝
连词：假设如果人而无信，不知其可也
连词：修饰地/着吾尝终日而思矣
代词通"尔"，你的而母立于兹
2. 何
疑问代词什么、哪里大王来何操
副词多么何其衰也
副词怎么（反问）徐公何能及君也
3. 乎
介词相当于"于"生乎吾前
语气助词吗、呢壮士，能复饮乎
词尾……的样子浩浩乎如冯虚御风
4. 乃
代词你的家祭无忘告乃翁
副词于是，就乃入见
副词竟然（转折）今其智乃反不能及
副词是，就是（判断）当立者乃公子扶苏
5. 其
人称代词他(的)/它(的)其闻道也固先乎吾
指示代词那、那些、其中的于乱石间择其一二扣之
语气副词大概（推测）其皆出于此乎
语气副词难道（反问）其孰能讥之乎
连词如果其若是，孰能御之
6. 且
连词而且且焉置土石
连词尚且（让步）臣死且不避
副词将要、将近不出，火且尽
7. 若
代词你、你们更若役，复若赋
连词如果若使烛之武见秦君
8. 所
助词构成"所"字结构道之所存，师之所存也
被动句式为……所……为仲卿母所遣
复音虚词所以：……的原因/用来……的师者，所以传道受业解惑也
9. 为
介词wèi给、替
介词wèi因为
介词wéi被（被动）
动词wéi做、是
语气助词wéi呢
10. 焉
兼词于之（在这里/那里）积土成山，风雨兴焉
代词哪里、怎么且焉置土石
语气助词了、啊于是余有叹焉
词尾……的样子盘盘焉，囷囷焉
11. 也
语气助词表判断（不译）城北徐公，齐国之美丽者也
语气助词表句中停顿师道之不传也久矣
12. 以
介词用、拿、凭借愿以十五城请易璧
介词因为不以物喜，不以己悲
连词来、用来（目的）作《师说》以贻之
连词而（并列/修饰）夫夷以近，则游者众
13. 因
介词趁着、凭借因击沛公于坐
介词通过因宾客至蔺相如门谢罪
连词于是、便相如因持璧却立
14. 于
介词在、从青，取之于蓝
介词比（比较）青于蓝
介词被（被动）见欺于王而负赵
介词对、向于其身也，则耻师焉
15. 与
介词和、跟吾孰与徐公美
连词和备他盗之出入与非常也
动词给予、结交蹇叔之子与师
语气助词通"欤"无乃尔是过与
16. 则
连词就、便（承接）故木受绳则直
连词如果（假设）入则无法家拂士
副词就是（判断）此则岳阳楼之大观也
17. 者
助词……的人/事/物楚人有涉江者
助词定语后置标志马之千里者
代词用于数词后（个、样）此数者用兵之患也
18. 之
人称代词他/她/它作《师说》以贻之
指示代词这、这些均之二策
动词到……去辍耕之垄上
助词的（定语的标志）古之学者必有师
助词取消句子独立性师道之不传也久矣
助词宾语前置标志何陋之有
助词音节助词（不译）顷之

四、阅读答题技巧

（一）非连续性文本阅读

1. 阅读策略（两种方法，择一而行）

方法一：先读题，再读文

先看主观题题干，明确阅读方向和重点

带问题快速浏览全文，圈画关键词

精准定位，对照选项回原文对比

方法二：先通读，再做题

通读全文，边读边圈画重点，明确每则材料核心

扫读题干，圈出关键词

基于全文理解，快速回文对比排除

2. 时间管理：整套控制在17-20分钟，不在个别题上过度纠结

3. 圈画关键词技巧

逻辑关联词：因为、所以、因此、但是、然而

绝对化词语：全部、必然、一定、根本、必须

范围程度词：部分、多数、几乎、仅、主要、最重要

时态词：已经、将、了、着、至今、未来

指代词：其、这、那、此

4. 客观选择题答题步骤（四步法）

一审：审题干关键词，明确选"正确"还是"不正确"

二找：在文中快速找出对应语句并标注

三比：将选项"切片"，圈出敏感词与原文比对

四判：根据比对结果确定答案

5. 常见设错陷阱

偷换概念/张冠李戴：将A事物特点安在B事物上

范围失当/以偏概全：将"部分"说成"全部"

无中生有：选项信息在原文中完全无依据

因果混乱：强加因果或因果倒置

绝对化表述："彻底解决""完全取代"等

6. 主观简答题模板

信息筛选与概括题：

一审题干限定词（角度、范围、时间）

二提取关键信息（抓关键句法 + 抓关键词法）

三归类合并整合

四校对：与题干和分值核对，分条作答（① ② ③）

比较材料异同题：

相同点："几则材料都认为/强调了……"

不同点："在……方面，材料一侧重于……，而材料二更关注……"

图表数据转换题：

看标题 -> 辨要素（横纵轴/表头） -> 抓关键（趋势/极值/对比） -> 精表述

句式示例："从……年到……年，……呈持续增长趋势，从……增长到……"

（二）小说阅读答题模板

小说答题核心四维度：人物、情节、环境、主题

1. 情节结构类

概括情节模板："何时何地何人做何事结果如何"

情节作用模板（含开头/中间/结尾）：

内容上：刻画人物性格 / 揭示主题

结构上：铺垫/伏笔/照应/推动情节

读者角度：设置悬念 / 引发思考

具体规范：

开头：设置悬念、渲染气氛、为下文做铺垫

中间：承上启下、推动情节发展

结尾：出人意料（情理之中）、升华主旨、留下想象空间

2. 人物形象类

分析角度：外貌、语言、动作、心理、神态 + 环境烘托、情节表现、他人对比

答题模板："他/她是一个……（总括性格/身份）的人。①从……（具体描写）可见其……；②从……（情节）可见其……"

3. 环境描写类（四维分析法）

环境本身：交代时间地点、渲染……气氛

人物：烘托人物心境、衬托性格

情节：推动情节发展、为下文……做铺垫

主题：暗示社会背景、深化主题

4. 标题作用类

内容：点明写作对象、概括核心情节

结构：作为全文线索贯穿全文

手法/情感：象征、一语双关、暗示主题

读者：设置悬念吸引阅读兴趣

5. 语言与技巧类

赏析句子模板："运用……手法（修辞/描写/表现）+ 写出了……（对象特点）+ 表达了……（人物情感）+ 具有……（艺术效果）"

（三）散文阅读答题模板

核心特征：形散神不散，答题关键是透过"形"抓住"神"。

1. 理解词句含义

三步法：表层义（字典义/指代义）-> 语境义（文中具体所指）-> 主旨义（象征意义、情感升华）

概括主旨模板："文章通过描写……，表达了作者……的情感，揭示了……的人生哲理/社会思考"

2. 句段作用题（内容 + 结构 双维度）

位置结构作用
文首开篇点题、总领全文、奠定感情基调、为下文铺垫
文中承上启下、过渡自然、前后呼应
文末卒章显志、深化中心、首尾呼应、言有尽而意无穷

3. 艺术手法鉴赏

答题模板："运用……手法 + 结合文本分析如何用 + 描绘……意象/景 + 表达……情感/哲理"

重点技巧：象征/托物言志（点明物的特征与志的关联）

4. 语言赏析

炼字：解释字义 + 描绘画面/动态 + 表达心境/情感

句式：分析长短句、整散句结合效果

5. 常用表现手法

类别具体术语
表达方式叙述（顺叙、倒叙、插叙）、描写、抒情、议论
表现手法借景抒情、情景交融、托物言志、象征、动静结合、虚实相生、白描、以小见大、抑扬、对比、渲染烘托、衬托
修辞手法比喻、比拟、借代、夸张、对偶、排比、设问、反问、反复、双关
语言风格平实质朴、含蓄隽永、清新自然、典雅华丽、幽默风趣、简洁明快
布局谋篇开门见山、首尾呼应、卒章显志、伏笔照应、层层深入、过渡铺垫、悬念、线索

6. 考场核心提醒

不动笔墨不读书：阅读时务必用笔圈画关键词句

答案在文中：大部分答案在原文中有依据或原型，学会"整合加工"

避免"贴标签"：不能只堆砌术语，必须结合原文具体内容

分点作答：使用①②③标注，先写结论/术语，再结合文本分析

五、高考议论文写作

（一）核心结构模板

"引议联结"法（高分首选）

**引**第1段引述材料，开篇入题，50-100字
**议**第2-4段分析材料，表明观点，并列/递进/对照展开
**联**第4-5段联系现实（社会热点/历史经验/个人成长）
**结**第5-7段强化论点，总结升华，回扣材料

五大常用结构模式

并列式分论点平行展开概念分析型
递进式是什么->为什么->怎么办深层思辨议题
正反对比式正面+反面论证有明确对立观点的题目
起承转合式逻辑严密，层层转折需深化立意的题目
引议联结式紧扣材料，联系现实任务驱动型/材料作文

全篇基本框架

引论（100-150字）：提材料、链接主题 -> 表态度、提出中心论点

本论（600-660字）：
  ├─ 分论点1（观点句+阐释句+事例句+分析句+总结句）
  ├─ 分论点2（同上）
  └─ 分论点3（同上）

结论（100-150字）：强化论点，总结升华，呼应开头
（二）主体段落写作"五步法"
① 观点句（分论点）-> 段首提出，紧扣中心
② 阐释句 -> 解释分论点，过渡衔接
③ 事例句 -> 精简叙述素材（详例/排比例/正反对比例）
④ 分析句 -> 因果/假设/对比/归纳分析
⑤ 总结句 -> 回扣观点，联系现实

分析句常用句式：

假设：如果不是……就没有……

因果：正因为……所以……

条件：有了……才有……

对比：反观……；与此相反……

（三）六种论证方法
举例论证事实胜雄辩精简叙述，避冗长
引用论证增强权威性名言+自己分析
对比论证突出是非二元作文必备
比喻论证化抽象为形象比喻引申类作文
因果论证揭示内在联系深层分析必备
假设论证反面强化论点与举例搭配使用
（四）拟标题技巧

对称式标题模板：

一元：唯有___，方能致远

二元并存：乘___之风，踏___征程

二元递进：___诚可贵，___价更高

二元对立统一：XX与XX齐飞，XX共XX一色

（五）素材运用新要求

禁用：司马迁、爱迪生等泛滥万能套例

标配：主题专属素材 + 古今新旧搭配（一类文标配）

趋势：素材必须紧扣题目核心关键词，体现思辨维度

单个事例控制在50-60字，多例排比高度凝练
